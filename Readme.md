# Behold the Bookstack!

A curated fullstack starter project assembled by Bookshop interactive.

### Client

- React (Generated by CRA2)
- React Router
- Storybook
- Styled Components
- React Apollo
- Enzyme

### Server

- Node
- Apollo Server
- LevelDB
- Babel Syntax Transpilation for ES2015+ Syntax
- Eslint
